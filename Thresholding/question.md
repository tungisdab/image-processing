Viết chương trình C++/Java/Python biến đổi phân ngưỡng cho ảnh input/output 256 cấp xám có đầu vào như sau:

- Dòng 1: nhập ngưỡng

- Dòng 2: nhập n

- n dòng tiếp theo là ma trận ảnh

Output

- Ma trận ảnh đã biến đổi

Ví dụ:

input

100

4 

50 150 200 50

75 180 30 220

160 40 90 120

210 70 250 25

output

0 255 255 0

0 255 0 255

255 0 0 255

255 0 255 0
